## Vim Commands

### Modes

* `i`: insert mode
* `ESC`: normal mode
* `v`: visual mode
* `V`: visual line mode

### Delete stuff

* `x`, `X`: delete character under, before cursor.
* `d`: delete something
* `y`: copy something

### Move around
* `b`, `w`, `e`, previous word (begining), next word, next word (end)
* `0`, `$`, begining and end of line
* `gg`, `G`, begining and end of file

### Misc

* `u`, Ctrl+R: undo, redo last command
* `:!<command>`, execute a command in shell
* `:r !<command>`, execute a command and insert its output in Vim
