# POSIX Extended attributes

Available attributes depend on the filesystem used.

## `lsattr`, `chattr`

* append only (a)
* no atime updates (A)
* compressed  (c)
* no  copy  on write (C)
* no dump (d)
* synchronous directory updates (D)
* extent format (e)
* immutable (i)
* data journalling (j)
* secure deletion (s)
* synchronous updates (S)
* no tail-merging (t)
* top of directory hierarchy (T)
* undeletable (u)
